# Encapsulation
* le principe d'encapsulation dit qu'un objet ne doit pas exposer sa représentation interne au monde exterieur
	* Monde Interne d'un objet: 
		* Atributs 
		* Methods
	* Monde Exterieur 
		* Autres Classes dans le même package
		* Classes dans d'autres Packages
